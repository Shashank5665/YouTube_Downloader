# YouTube_Downloader
YouTube downloader pc application built using python.

Libraries used : <br/>
=> Tkinter <br/>
=> Pytube <br/>
<br/>
To install the libraries, type these commands in your terminal :
<br/>
```pip install tk```
<br/>
```pip install pytube```
<br/>
<br/>
You are ready to go...
