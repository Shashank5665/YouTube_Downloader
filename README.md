# YouTube_Downloader
YouTube downloader pc application built using python.

<h2>Prerequisites</h2>
Libraries used : <br/>
-> Tkinter <br>
-> Pytube <br>
<br>
<h2>How to run the script</h2>

`pip install tk`
<br>
`pip install pytube`
<br>
`python YouTube_Downloader.py`
